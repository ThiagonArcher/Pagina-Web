# Mi Tienda Escolar

Este proyecto es un catálogo de productos hechos por estudiantes. Contiene información de 4 productos: Taza personalizada, Playera estampada, Cuaderno creativo y Lápices de colores.

## Estructura del proyecto

- `index.html` → Contiene la estructura de la página web con HTML5, utilizando marcado semántico: `<header>`, `<section>`, `<article>`, `<footer>`.
- Imágenes de productos: `taza.jpg`, `playera.jpg`, `cuaderno.jpg`, `lapices.jpg`

## Características

- Encabezado con el nombre de la tienda y eslogan
- Catálogo con 4 productos con nombre, imagen, descripción y precio
- Pie de página con datos de contacto
- Uso de atributos `alt` en todas las imágenes para accesibilidad

## Control de versiones

Se utilizó **Git** para llevar un registro de los cambios del proyecto:

- **Primer commit:** Estructura básica de HTML
- **Segundo commit:** Agregar productos al catálogo
- **Tercer commit:** Agregar imágenes y correcciones

### Beneficios de usar Git

- Permite **volver a versiones anteriores** si hay errores
- Facilita la **colaboración con otras personas**
- Mantiene un **historial claro de todos los cambios** en el proyecto
